# 히스토린더
2019 Smarteen App+ Chellange 생활부문 참가
재미있고 잘 알려지지 않은 역사를 달력 형태로 쉽게 볼 수 있는 안드로이드 앱


## 01. 프로그램 특징
* 날짜 클릭 시 해당 날짜의 역사 사건을 볼 수 있음
* 관심있는 역사 사건을 설정하여 관리
* 음성 알람 - 사용자가 좋아하는 시대의 역사를 시간순으로 들려줌
* 상단바 알림 - 랜덤으로 역사 알림을 띄움


## 02. 사용 기술
* Android(Java)
* SQLite
